To implement this scenario in SoPLang, we need to create a script that differentiates between weekdays and weekends, opens the curtains and provides weather notifications with clothing recommendations on weekdays, and only turns on the lights on weekends. Here's how you can write this script:

```sop
loop (24 HOUR)
{
    wait until ((#clock).hour == 9)

    // Check if it's a weekday (Monday to Friday)
    if ((#clock).weekday != "Saturday" and (#clock).weekday != "Sunday")
    {
        // Open the curtains
        (#Shade).on()

        // Get weather information
        temp = (#weather_builtin_big_thing).temp()
        humid = (#weather_builtin_big_thing).humid()
        air_quality = (#AirPurifier).airPurifierFanMode()

        // Prepare weather notification and clothing recommendation
        if (temp < 10)
        {
            clothing = "It's cold outside. Wear a warm coat."
        }
        else if (temp >= 10 and temp < 20)
        {
            clothing = "It's cool outside. A jacket should be enough."
        }
        else
        {
            clothing = "The weather is warm. Light clothing is fine."
        }

        if (humid > 70)
        {
            clothing = clothing + " It's quite humid, consider breathable fabrics."
        }

        if (air_quality == "high")
        {
            clothing = clothing + " The air quality is poor, consider wearing a mask."
        }

        // Output notification
        (#Speaker).tts("Good morning! " + clothing)
    }
    else
    {
        // On weekends, just turn on the lights
        (#Light).on()
    }
}
```

This script will:
- Run every 24 hours.
- Wait until 9 AM.
- Check if it's a weekday or weekend.
- On weekdays, it opens the curtains, retrieves weather data, and provides clothing recommendations based on temperature, humidity, and air quality.
- On weekends, it simply turns on the lights without providing any notifications.
